# powerbi-car-sales-dashboard

This is a Power BI dashboard created to analyze car sales data and answer basic business questions.

## What this dashboard shows
- Total units sold by car brand
- Total revenue generated by car models
- Number of available variants (distinct models)
- Brand-wise market value using bar chart
- Top 5 models by revenue for each brand
- Market share percentage of brands
- Sales trend over time

## Key Features
- KPI cards for Units Sold, Revenue, and Variants
- Interactive brand selection using logos
- Charts for revenue, market share, and sales trend
- Dynamic image update based on selected brand

## Tools Used
- Power BI Desktop
- DAX (basic calculations)
- Excel / CSV data

## Purpose
This project was built as a hands-on learning exercise to understand Power BI dashboard design, data modeling, and business analysis.
